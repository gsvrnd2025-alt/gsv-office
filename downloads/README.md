# GSV Office Client Downloads Directory

Place compiled clients and extension files here to make them downloadable from the web app interface:
- `GSVOffice-Setup.exe` (Windows Installer)
- `GSVOffice-Portable.exe` (Windows Standalone Portable)
- `GSVOffice-Mac.dmg` (macOS Desktop Client DMG)
- `GSVOffice-Android.apk` (Android APK) - Note: To generate the real Android APK, open the `frontend/android` folder in Android Studio and click Build -> Build Bundle(s)/APK(s) -> Build APK. Copy the resulting `app-debug.apk` or `app-release.apk` here as `GSVOffice-Android.apk`.
